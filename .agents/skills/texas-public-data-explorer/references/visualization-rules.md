# Visualization Rules

- Use line charts for time series.
- Use bar charts for ranked categories.
- Use maps/geography blocks when geography fields exist.
- Use tables for grouped results and inspectability.
- Use metric cards for totals and key stats.
- Use summary cards for plain-English explanation.
- Always include SourceMethodBlock.
- Avoid misleading charts: label axes, show units, preserve context, and mention caveats.
