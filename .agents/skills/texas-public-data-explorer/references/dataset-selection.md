# Dataset Selection Rules

1. Search approved catalog first.
2. Prefer official Texas city/state public portals.
3. Prefer datasets with clear source, license/terms, date fields, geography fields, and update metadata.
4. Prefer datasets that can answer the user question with aggregation.
5. If confidence is low, show candidates and ask the user to choose.
6. Do not use sources behind authentication.
7. Do not invent dataset names or URLs.
8. Use sample fallback only for demo reliability and label it clearly.
