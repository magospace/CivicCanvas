# Miro Export Rules

Miro export converts a safe Texas Data Canvas dashboard into a presentation/workshop board.

Rules:

1. Generate MiroExportSpec from an existing CanvasDocument.
2. Include source/method/caveats frame.
3. Do not export hidden/sensitive fields.
4. Do not write to Miro automatically.
5. Ask for user confirmation before board creation.
6. Prefer aggregate tables and images.
7. Label sample/demo data if used.
8. Keep Texas Data Canvas as the data governance layer.
