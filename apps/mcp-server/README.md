# Texas Public Data MCP Server

Typed scaffold for the future `texas-public-data-mcp` server.

P0/P1 intentionally stops at package structure and typed tool inventory. P4 will implement the MCP runtime and validated tools.

## Commands

```bash
pnpm --filter @texas-data-canvas/mcp-server typecheck
pnpm --filter @texas-data-canvas/mcp-server build
```
