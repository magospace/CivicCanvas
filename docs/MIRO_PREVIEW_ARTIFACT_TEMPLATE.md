Use this template when manually saving or sharing the preview-only `MiroExportSpec` JSON generated by Texas Data Canvas. This template does not create a Miro board and is not release evidence unless Task 35 later incorporates it into a full release-evidence refresh.

## Artifact Boundary

Current implementation status:

- The app generates preview-only `MiroExportSpec` JSON.
- No Miro OAuth flow exists.
- No access token is requested or stored.
- No board ID is returned by the app.
- No Miro board write occurs.
- The preview must include source/method/caveat content.

## Required Caveats If Mentioned

Use wording like:

> Texas Data Canvas can generate a validated preview-only MiroExportSpec JSON handoff. It does not authenticate with Miro, request OAuth, create a board, update a board, or store Miro tokens.

Do not say:

- "Exports directly to Miro"
- "Creates a Miro board"
- "Syncs with Miro"
- "Uses Miro OAuth"
- "Writes collaboration boards"
